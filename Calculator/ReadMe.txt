# Simple Python Calculator

This is a simple Python program that acts as a calculator. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## How to Use

1. Clone or download the repository.
2. Run the `calculator.py` file using Python.
3. Follow the on-screen instructions to select the desired operation and input numbers accordingly.

## Operations

The calculator supports the following operations:

- **Addition (+)**
- **Subtraction (-)**
- **Multiplication (*)**
- **Division (/)**
